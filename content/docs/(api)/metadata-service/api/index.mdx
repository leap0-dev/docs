The Metadata Service API is available inside the sandbox at `http://169.254.169.254`.

Use it to read sandbox metadata from inside the guest, including environment details and service state exposed by Leap0.

## Base URL

`http://169.254.169.254`

## Authentication

The metadata service uses IMDSv2 semantics from inside the sandbox.

## What It's For

- Read metadata about the current sandbox
- Inspect guest-visible Leap0 state
- Access sandbox-local metadata flows without calling the public API
